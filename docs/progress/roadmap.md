# Roadmap

## Phase 1 — MVP (offline, no AI)
Goal: working app that can teach from Murphy Blue without Claude API.

- Placement test → detect CEFR level
- PDF import → 145 units in SQLite
- Curriculum → sequential unlock
- Chapter view → read grammar explanation
- Exercise view → answer questions, see score
- Progress → track completion per unit

## Phase 2 — AI Layer
Goal: enhance with Claude for users who add API key.

- Settings screen (add/remove Claude API key)
- AI explanations: "Ask Claude about this unit"
- AI exercise generation: auto-generate extra practice
- Adaptive hints: wrong answer → Claude explains why

## Phase 3 — Polish & Distribution
Goal: shareable .dmg.

- App icon + branding
- Onboarding polish (animations, better UX)
- Dark mode
- Daily streak + goal tracking
- Package as .dmg (macOS) + .exe (Windows)
- GitHub Releases page

## Phase 4 — Future ideas
- Support other textbooks (Headway, Cambridge Grammar in Context)
- Spaced repetition for vocabulary
- Listening exercises (audio playback)
- Export progress as PDF report
